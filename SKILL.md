# 词汇卡片生成器

## 功能概述

将用户提供的单词列表转换为美观的交互式单词卡片。每个卡片支持：

- ✨ **翻转效果** - 点击卡片查看背面解释
- 🔊 **发音功能** - 点击发音按钮朗读单词
- 📱 **响应式设计** - 适配不同屏幕尺寸
- 💾 **自动保存** - 每次生成唯一时间戳文件，避免覆盖

## 输入格式

支持两种分隔格式，每行一个单词：

```
格式1: 单词 | 解释
apple | 苹果，一种常见的水果

格式2: 单词, 解释
banana | 香蕉，热带水果
```

**示例输入：**
```
apple | 苹果，一种常见的水果
banana | 香蕉，热带水果，黄色弯曲
orange | 橙子，柑橘类水果，富含维生素C
```

## 使用方法

1. 直接发送包含单词和解释的文本列表
2. 系统会自动解析并生成HTML文件
3. 在浏览器中打开生成的HTML文件即可使用

## 输出文件

- 文件名格式：`vocabulary-cards-YYYYMMDD_HHMMSS.html`
- 包含完整的HTML、CSS和JavaScript代码
- 无需网络连接，可直接在浏览器中打开使用

## 卡片特性

### 视觉设计
- 现代简约的卡片样式
- 正面：单词大字体显示 + 发音按钮
- 背面：解释文本
- 优雅的翻转动画效果

### 交互功能
- 点击卡片任意位置翻转
- 点击🔊按钮朗读单词
- 悬停效果提示

### 技术实现
- 使用CSS 3D transform实现翻转效果
- 使用Web Speech API实现发音功能
- 纯HTML/CSS/JavaScript，无外部依赖

## 注意事项

- 确保输入格式正确（使用 | 或 , 分隔）
- 每行只包含一个单词及其解释
- 生成的文件保存在当前工作目录
- 支持中英文解释
